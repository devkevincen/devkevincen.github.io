对 HTML 语义化的理解#

🟠 频率：适中

HTML 语义化是指在编写 HTML 文档时，使用恰当的标签，以及正确的结构来描述文档的内容、结构和意义。比如 header 就是放一些头部的标签，footer 就是放一些底部的标签。这么做有几点好处：

  • 对机器友好，更适合搜索引擎的爬虫爬取有效信息，有利于 SEO。
  • 对开发者友好，使用语义类标签增强了可读性，结构更加清晰，开发者能清晰地看出网页的结构，便于开发与维护。

SEO(Search Engine Optimization): 搜索引擎优化，为了让用户在搜索和网站相关的关键词的时候，可以使网站在搜索引擎的排名尽量靠前，从而增加流量。

常见的语义化标签有哪些#

🔴 频率：较少

<header> 头部
<nav> 导航栏
<article> 主要内容
<aside> 侧边栏
<footer> 底部

行内元素、块级元素、行内块元素的的特点？分别有哪些#

🟠 频率：适中

行内元素：

  • 一行可以显示多个
  • 不可以设置宽高
  • 宽度和高度默认由内容撑开
  • padding 左右可以设置，上下设置不占位 （浏览器中可以去验证一下）
  • margin 左右可以设置，上下设置不生效 （浏览器中可以去验证一下）
  • 行内元素有 span a b i 等等

块级元素：

  • 独占一行
  • 可以设置宽高
  • 宽度默认是父元素的宽度（父元素宽度 100%），高度默认由内容撑开
  • margin、padding 上下左右都有效
  • 块级元素有 h1-h6 div p 等等

行内块元素：

  • 一行可以显示多个
  • 可以设置宽高
  • 宽度和高度默认由内容撑开
  • margin、padding 上下左右都有效
  • 行内块元素有：img button input 等等

空(void)元素有那些？#

🔴 频率：较少

空元素：没有内容的 HTML 元素。空元素是在开始标签中关闭的，即空元素没有闭合标签

常见的有：<br>、<hr>、<img>、<input> 等

HTML5 有哪些更新#

🟠 频率：适中

  • 语义化标签 HTML5 中引入了语义化标签，可以为页面提供更多的语义化信息，提高网页的可读性和可访问性。而且搜索引擎也能更准确地索引网页内容，提供更好的搜索结果。比如：header 定义文档的的头部区域、nav 定义文档的导航链接部分、article 定义文档的内容区域、footer 定义文档的底部区域等等
  • input 的类型：color 拾色器、date 日期、month、week、time 等等
  • input 的属性：required、placeholder、autofocus 等等
  • 新增了视频标签 video 和音频标签 audio
  • HTML5 提供了两种在客户端存储数据的新方法，分别为 localStorage 和 sessionStorage。 其中 localStorage 关掉网页之后数据仍然存在，sessionStorage 关掉当前网页数据就没有了
