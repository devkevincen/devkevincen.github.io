优秀的提示词：#

  • 【立角色】：引导 AI 进入具体场景，赋予其行家身份
  • 【述问题】：告诉 AI 你的困惑和问题，以及背景信息
  • 【定目标】：告诉 AI 你的需求，希望达成的目标
  • 【补要求】：告诉 AI 回答时注意什么，或者如何回复

不好的提示词：我想去海南玩，请你帮我做一份旅行攻路

好的提示词：假如你是导游，我要到海南游玩，预算一万元，旅行人数三个人，行程7天，请称我做一份旅行政略，请注量：我不喜欢行程太紧凑，我不喜欢网红景点，更喜欢有文化底蕴的景点，另外，推荐景点请附上各个景点的价格

提示词模版：#

  1. 将提示词提炼成模板
  2. 实现提示词的复用、版本管理、动态变化等

以上面的好的提示词举例：

假如你是导游，我要到 {place} 游玩，预算 {budget}，旅行人数 {numberofpeople} 个人，行程 {numberofdays} 天，请称我做一份旅行政略，请注量：我不喜欢 {dontlike}，更喜欢 {like}，另外，{additional}。

将一条固定的提示词，进行提炼，来实现提示词的复用，即为提示词模版。像极了一个函数，传入所需要的参数，返回一定内容， y = f(x1, x2, ...)

Prompt Templates 提示此模版#

字符串模版 PromptTemplate#

from langchain.prompts import PromptTemplate
prompt_template = PromptTemplate.from_template("你是一位起名大师，帮我起{number}个具有{country}特色的{sex}名字")
prompt_template.format(number=1, country='中国', sex='女孩')

很像 JS 中的模版字符串，对吧 🤔

对话模版 ChatPromptTemplate#

from langchain_core.prompts import ChatPromptTemplate

chat_template = ChatPromptTemplate.from_messages([
    ("system", "你是一个起名大师，你的名字叫{name}"),
    ("human", "你好{name}，你的感觉如何？"),
    ("ai", "你好！我状态非常好！"),
    ("human", "{user_input}"),
])

chat_template.format_messages(name="小栈大师", user_input="你叫什么名字？")

AI 将会回答你的最后一个问题，即这里的 user_input，其余内容将作为上下文 (context) 给到 AI 作为背景信息，以提高 AI 回答的准确度

当然 LangChain 也非常灵活，从上面的图片中你可以看到 chat_template.format_messages 后是一个 list，每一项分别是 SystemMessage、HumanMessage、AIMessage，你可以亲自通过创建这些类的对象，比如：

from langchain.schema import AIMessage, HumanMessage, SystemMessage

sy = SystemMessage(
    content="你是一个起名大师",
    additional_kwargs={"大师姓名": "小栈大师"}
)

hm = HumanMessage(
    content="大师叫什么名字",
)

am = AIMessage(
    content="我叫小栈大师",
)

[sy, hm, am]

运行结果和上面是一样的

当然，LangChain 已经将三个默认的三个角色进行模版化了，分别为 AIMessagePromptTemplate、SystemMessagePromptTemplate、HumanMessagePromptTemplate、ChatMessagePromptTemplate

from langchain.prompts import AIMessagePromptTemplate
from langchain.prompts import SystemMessagePromptTemplate
from langchain.prompts import HumanMessagePromptTemplate
from langchain.prompts import ChatMessagePromptTemplate

prompt = "你好，{name}"

ai_message_prompt = AIMessagePromptTemplate.from_template(template=prompt)
ai_message_prompt.format(name="小栈大师")

sys_message_prompt = SystemMessagePromptTemplate.from_template(template=prompt)
sys_message_prompt.format(name="Kevin")

hum_message_prompt = HumanMessagePromptTemplate.from_template(template=prompt)
hum_message_prompt.format(name="Cen")

# 自定义角色
custom_message_prompt = ChatMessagePromptTemplate.from_template(role="程序员", template=prompt)
custom_message_prompt.format(name="程序员")

# 代码注释生成器：解放双手，帮你写函数注释

from langchain.prompts import StringPromptTemplate

def hello():
    print("hello, coderstack")
    return "coderstack"

prompt = """你是一位极其有经验和天赋的程序员，我将给你一个函数名称，你需要按照如下格式，输出这段代码的源代码、注释。
函数名称：{function_name}
源代码：
{source_code}
注释：
"""

import inspect

def get_source_code(name):
    return inspect.getsource(name)

class CustomPrompt(StringPromptTemplate):
    def format(self, **kwargs) -> str:
        # 获取源代码
        source_code = get_source_code(kwargs["name"])

        prompt_to_ai = prompt.format(
            function_name=kwargs["name"].__name__,
            source_code=source_code
        )

        return prompt_to_ai

cp = CustomPrompt(input_variables=["function_name"])
prompt_after_format = cp.format(name=hello)

print(prompt_after_format)
print("====== 以上为传给 LLM 的 prompt ======")

from langchain_openai import OpenAI
import os

# 从你电脑中的环境变量中读取 api_key (Mac 的环境变量配置通常在 ~/.zshrc 文件中)
api_key = os.getenv("OPENAI_API_KEY")

llm = OpenAI(
    model="gpt-3.5-turbo-instruct",
    temperature=0,
    openai_api_key=api_key
)

result_from_ai = llm.invoke(prompt_after_format)
print(result_from_ai)

因此，我们可以看到，使用自定义模版可以比较轻松地写出一个自己想要的 prompt 模版，并且实现一个小的应用：代码注释生成器

组合提示词 Pipeline prompts & Final prompt#

  • Final prompt：最终返回的提示词模版
  • Pipeline prompts：组成 Final prompt 的提示词模版
from langchain_core.prompts import PipelinePromptTemplate, PromptTemplate

full_template = """{introduction}
{example}
{start}"""
full_prompt = PromptTemplate.from_template(full_template)

introduction_template = """你正在冒充{person}"""
introduction_prompt = PromptTemplate.from_template(introduction_template)

example_template = """这是一个互动的例子：
Q: {example_q}
A: {example_a}"""
example_prompt = PromptTemplate.from_template(example_template)

start_template = """现在，来真的吧！
Q: {input}
A:"""
start_prompt = PromptTemplate.from_template(start_template)

input_prompts = [
    ("introduction", introduction_prompt),
    ("example", example_prompt),
    ("start", start_prompt),
]

pipeline_prompt = PipelinePromptTemplate(
    final_prompt=full_prompt, pipeline_prompts=input_prompts
)

print("组合后模版的可传递参数 ", pipeline_prompt.input_variables, "\n")

print(
    pipeline_prompt.format(
        person="小栈",
        example_q="你最喜欢的brand是什么",
        example_a="Apple",
        input="你最喜欢的App是什么",
    )
)

这样的写法看上去比较复杂，但是当我们遇到比较大的 prompt 模版，并且可以通过复用之前写过的 prompt 模版时，这样子的操作可以提升效率
